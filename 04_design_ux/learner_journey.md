# 학습자 여정

## 초등 학생

1. AI가 맞히거나 틀리는 장면을 본다.
2. 내가 데이터를 추가하면 결과가 바뀐다는 것을 경험한다.
3. AI가 틀린 이유를 말로 설명한다.
4. 생활 문제에 적용해 본다.

## 중등 학생

1. 같은 문제를 여러 알고리즘으로 풀어 본다.
2. 결과 차이를 표로 비교한다.
3. 알고리즘의 장단점을 토론한다.
4. 상태, 규칙, 조건, 데이터 개념을 연결한다.

## 고등·대학 학생

1. 파라미터를 조작한다.
2. 손실, 정확도, 경계, Q값 등을 해석한다.
3. 코드 또는 수식과 연결한다.
4. 캡스톤 문제를 설계한다.

## 성인 학습자

1. AI가 할 수 있는 일과 없는 일을 구분한다.
2. 내 업무 문제를 작게 나눈다.
3. AI를 초안·정리·분석 보조로 쓴다.
4. 결과를 검증하고 책임 범위를 정한다.
