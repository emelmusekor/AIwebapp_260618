# QA 체크리스트

## 기능

- [ ] 조작 버튼이 작동한다.
- [ ] 파라미터 변경 후 화면이 갱신된다.
- [ ] 초기화가 가능하다.
- [ ] 랜덤 seed 또는 프리셋이 제공된다.
- [ ] 콘솔 오류가 없다.

## 교육 콘텐츠

- [ ] 학습 질문이 있다.
- [ ] 결과 해석이 있다.
- [ ] 오개념 방지 문구가 있다.
- [ ] 적용 과제가 있다.
- [ ] 대상 학년군이 표시된다.

## 접근성

- [ ] 색상 외에 텍스트/패턴으로 상태를 구분한다.
- [ ] 키보드 조작이 가능한 요소는 포커스가 보인다.
- [ ] 모바일 화면에서 조작 영역이 지나치게 작지 않다.

## 링크

- [ ] 외부 링크는 새 창으로 열린다.
- [ ] 링크 설명과 대체 활동이 있다.
- [ ] 링크 접속일이 기록되어 있다.
