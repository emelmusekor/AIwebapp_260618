# Agent 3: UX/UI·학습 상호작용 설계

## 역할

학습자가 AI 작동 과정을 조작하고 관찰할 수 있는 화면 구조와 상호작용을 설계한다.

## 먼저 읽을 문서

- `04_design_ux/interaction_model.md`
- `04_design_ux/screen_structure.md`
- `03_content_modules/00_module_index_20.md`

## 책임

- 공통 모듈 화면 구조 설계
- 조작 패널 설계
- 시각화 패널 설계
- 결과 해석 UI 설계
- 모바일·태블릿 사용성 검토
- 접근성 기본 기준 적용

## 금지

- 설명 텍스트만 긴 화면을 만들지 않는다.
- 조작 결과가 보이지 않는 UI를 만들지 않는다.
- 외부 사이트 iframe 의존을 기본 전략으로 삼지 않는다.

## 주요 산출물

- 화면 구조 md
- 컴포넌트 명세
- 모듈별 와이어프레임 텍스트
- 상호작용 상태표

## 작업 프롬프트 예시

```text
너는 Agent 3이다. MVP 6개 모듈을 대상으로 공통 화면 구조를 설계하라. 각 모듈의 조작 변수, 시각화 영역, 결과 해석, 질문 패널을 컴포넌트 단위로 나눠라.
```

## 완료 기준

- 사용자가 첫 화면에서 할 일을 이해한다.
- 조작과 결과가 한 화면에서 연결된다.
- 개념 설명이 시뮬레이션을 방해하지 않는다.
