# 비지도학습 기획

## 핵심 질문

정답이 없을 때 AI는 데이터 속 구조를 어떻게 찾는가?

## 포함 모듈

- K-Means 군집 지도
- DBSCAN형 밀도 군집 비교
- 임베딩 공간 탐험
- 우리 반 취향 지도

## 학습 목표

- 군집화는 정답 맞히기가 아니라 구조 발견임을 이해한다.
- 거리, 중심점, K값의 의미를 설명한다.
- K값이 너무 작거나 크면 해석이 달라짐을 관찰한다.
- 임베딩 공간에서 비슷한 데이터가 가까이 놓일 수 있음을 이해한다.

## 주요 상호작용

| 조작 | 결과 |
|---|---|
| 점 추가 | 군집 배정 변화 |
| K값 변경 | 군집 수와 해석 변화 |
| 중심점 초기화 | 다른 결과 가능성 관찰 |
| 거리 기준 변경 | 군집 형태 변화 |
| 임베딩 카드 이동 | 유사도 개념 체험 |

## 실제 적용

- 학생 취향 군집
- 고객 유형 분류
- 문서·이미지 검색
- 추천 시스템 기초
- 이상치 탐지

## 오개념 방지

- 군집 결과는 정답이 아니라 해석 후보다.
- K-Means는 둥근 군집에는 강하지만 복잡한 모양에는 약하다.
- 가까운 벡터가 항상 인간이 느끼는 의미와 일치하지는 않는다.
