# 신경망과 CNN 기획

## 핵심 질문

신경망은 입력을 어떻게 계산하고, CNN은 이미지를 어떻게 부분 특징으로 보는가?

## 포함 모듈

- 퍼셉트론과 OR/AND/XOR
- 작은 신경망 학습률 실험
- 0–9 숫자 인식 신경망
- CNN 필터와 특징맵
- Grad-CAM형 판단 근거 보기

## 학습 목표

- 가중치, 편향, 활성화 함수의 의미를 설명한다.
- 선형분리와 XOR 문제를 통해 은닉층 필요성을 이해한다.
- 학습률과 과적합을 조작해 본다.
- CNN이 작은 필터로 특징을 추출함을 관찰한다.
- 숫자 인식 결과가 확률적 예측임을 이해한다.

## 숫자 인식 모듈 요구사항

| 기능 | 설명 |
|---|---|
| Canvas 입력 | 사용자가 0–9 숫자를 그린다 |
| 전처리 표시 | 28x28 흑백 이미지로 변환되는 과정을 보여준다 |
| 예측 표시 | 각 숫자별 confidence bar 표시 |
| 오답 분석 | 왜 3과 8, 1과 7을 헷갈렸는지 시각 특징으로 설명 |
| 재학습 확장 | MVP 이후 사용자 예시 추가 학습 기능 검토 |

## 실제 적용

- 우편번호 인식
- 시험지 자동 채점
- 교통 표지판 인식
- 의료 영상 보조 진단의 한계
- 불량품 탐지

## 오개념 방지

- CNN은 이미지를 사람처럼 이해하지 않는다.
- 숫자 인식 결과는 확률적 예측이다.
- 배경, 기울기, 일부 가려짐에 취약할 수 있다.
