# 지도학습 기획

## 핵심 질문

정답이 붙은 예시를 보고 새 데이터를 맞히는 AI는 어떻게 작동하는가?

## 포함 모듈

- 내가 AI를 가르치는 이미지 분류
- 2D 결정경계 실험실
- 회귀와 예측 모델
- 데이터 품질과 일반화 실험

## 학습 목표

- 라벨과 학습 데이터의 의미를 설명한다.
- 훈련 데이터와 테스트 데이터의 차이를 이해한다.
- 데이터가 부족하거나 치우치면 모델이 틀릴 수 있음을 관찰한다.
- 같은 데이터라도 알고리즘에 따라 판단 경계가 달라짐을 비교한다.

## 주요 상호작용

| 조작 | 결과 |
|---|---|
| 데이터 점 추가 | 결정경계 변화 |
| 라벨 변경 | 모델 판단 변화 |
| 알고리즘 선택 | KNN, 선형, 트리형 경계 비교 |
| 노이즈 추가 | 정확도와 일반화 변화 |
| train/test 비율 변경 | 과적합 관찰 |

## 실제 적용

- 교실 물건 분류
- 바른 자세 알림
- 민원 문장 분류
- 합격/불합격 예측의 위험성 토론
- 불량품 이미지 분류

## 오개념 방지

- 높은 정확도가 항상 좋은 모델을 뜻하지 않는다.
- 훈련 데이터에만 잘 맞는 모델은 실제 상황에서 약할 수 있다.
- AI가 틀린 원인은 알고리즘뿐 아니라 데이터 수집 방식일 수 있다.
