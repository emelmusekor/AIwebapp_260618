# 제품 범위

## 제품 형태

웹 기반 인터랙티브 AI 학습 플랫폼.

## 핵심 사용자

- 초등 정보·AI 교육 교사
- 중등 정보 교사
- 대학 AI 입문 강의자
- 성인 AI 리터러시 교육 담당자
- 학생 개인 학습자

## MVP 범위

MVP는 20개 전체 모듈 중 핵심 6개를 먼저 만든다.

| MVP 모듈 | 핵심 기능 | 구현 난도 |
|---|---|---|
| 숫자 인식 신경망 | Canvas 숫자 입력, 예측, confidence, 오답 분석 | 높음 |
| 2D 지도학습 | 점 찍기, 라벨 지정, 결정경계 변화 | 중간 |
| K-Means | K값 조절, 중심점 이동, 반복 애니메이션 | 중간 |
| 미로 탐색 | BFS, DFS, Dijkstra, A* 비교 | 중간 |
| 강화학습 GridWorld | 보상, Q-table, policy 시각화 | 높음 |
| 토큰·Attention | 텍스트 토큰화, attention 연결선/히트맵 | 중간 |

## 확장 범위

- CNN 필터와 feature map
- RNN/LSTM gate
- Diffusion 노이즈 제거
- 전문가 시스템
- CSP 자리 배치
- Agent workflow
- micro:bit/Edge Impulse 연계 안내

## 제외 또는 보류

| 항목 | 이유 |
|---|---|
| 실시간 외부 API 기반 생성형 AI | 비용, 개인정보, 네트워크 의존성 |
| 대규모 딥러닝 학습 | 브라우저 성능 한계 |
| 카메라·마이크 기본 활성화 | 초등 교육 개인정보 리스크 |
| 모든 외부 사이트 임베드 | 네트워크와 CORS 문제 |

## 성공 기준

- 교사가 1차시 수업에서 바로 하나의 모듈을 사용할 수 있다.
- 학습자가 조작 전후 결과 차이를 설명할 수 있다.
- 각 모듈에 오답·한계·적용 과제가 포함된다.
- 개발자가 문서만 보고 모듈 구현을 이어갈 수 있다.
